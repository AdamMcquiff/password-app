# Password App

> API and Web App for CET324. Built as a full-stack JavaScript application with Node.js, Express, Postgres and Vue.js.

## Web App

To setup and run the web app:

* `cd webapp`
* `npm install`
* `npm run dev` OR `npm run build`

## API

To setup and run the API:

* `cd api`
* `npm install`
* `npm run start:dev`
